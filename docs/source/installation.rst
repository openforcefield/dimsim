Installation
============

Installing from source
----------------------

To install dimsim from source, clone the repository and install with pip:

.. code-block:: bash

    git clone https://github.com/lilyminium/dimsim.git
    cd dimsim
    pip install -e .

Development installation
------------------------

For development, install with the dev extras:

.. code-block:: bash

    pip install -e ".[dev]"

This will install all dependencies needed for development, including testing and documentation tools.

Requirements
------------

dimsim requires Python 3.9 or later.
